# C.A.R.E — Complaint And Resolution Engine

C.A.R.E is a complaint management system designed to allow users to submit complaints and track their resolution status.

## Features
- User complaint submission
- Complaint tracking
- Resolution management
- Full-stack architecture

## Tech Stack
- Frontend
- Backend
- Java / Spring Boot
- HTML / CSS / JavaScript

## Status
Work in progress project focused on complaint handling and resolution workflow.
